# Atlas

Use this skill before touching an unfamiliar repository.

- Start with `rg --files`, manifest reads, and local instruction files.
- Identify runtime entry points, generated folders, test commands, and ignored
  paths.
- Return a map with enough detail to choose the next edit safely.
- Avoid speculative architecture summaries without file evidence.
