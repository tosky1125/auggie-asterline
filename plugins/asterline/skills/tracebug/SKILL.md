# Tracebug

1. State at least three plausible hypotheses.
2. Reproduce the failure with the smallest command or scenario.
3. Gather evidence that separates hypotheses.
4. Lock the defect with a regression check when practical.
5. Apply the smallest fix that explains the observed failure.
6. Verify through the real user-facing path, not only a helper function.
