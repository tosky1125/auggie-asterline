# Runbook

Use this skill to carry an approved plan to completion.

1. Confirm the active plan, constraints, and current repository status.
2. Create or refresh a task list with one active item.
3. Make narrow edits in the order that reduces risk fastest.
4. Record evidence as each acceptance criterion becomes true.
5. Rerun the relevant checks after every material fix.
6. End with a concise status, changed files, checks run, and remaining risk.
